## What This Is

Standard AI agents fail silently under adversarial pressure. They degrade into
policy violations while continuing to operate — presenting no outward signal
that their internal logic has been compromised.

The Constitutional Agent SDK eliminates this. It wraps any standard generative
model inside a **Viability Kernel** — a deterministic constitutional boundary
that the agent cannot cross without ceasing to operate.

If constitutional integrity cannot be maintained, the SDK executes a
**Constitutional Halt**: a designed, safe transfer of authority back to human
operators, with a complete deterministic audit log.

This is not a training approach. It is Alignment-by-Architecture.

intent (Phenotype) to TypeScript hard constraints (Genotype).

Tracks causal saturation `ρ = η/η_max` in real time.

### 3. Constitutional Halt API
